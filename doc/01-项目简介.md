# 租房避雷App - 项目简介

## 1. 产品概述

**产品名称**：租房避雷App

**产品定位**：租房信息查询平台，帮助租客（大学生、职场人）快速了解周边租房市场的小区和房屋真实情况。

**核心价值**：
- **信息透明**：真实用户评价，避免租房踩坑
- **便捷查询**：快速了解小区详情、老化程度、卫生问题、周边服务
- **防坑指南**：提供中介黑幕、押金问题、房源真实性等避雷信息

## 2. 目标用户

| 用户类型 | 描述 | 需求 |
|---------|------|------|
| 租客（大学生） | 在校学生，预算有限，关注性价比 | 寻找安全、便宜、交通便利的房源 |
| 租客（职场人） | 上班族，关注通勤和生活便利性 | 寻找位置好、环境舒适的房源 |
| 管理员 | 平台运营人员 | 内容审核、数据管理、系统配置 |

## 3. 核心功能

### 3.1 MVP功能（P0）
- 用户注册登录
- 首页展示推荐小区
- 小区搜索与多维度筛选
- 小区详情展示
- 评论发布与浏览
- 个人信息管理
- 小区管理后台
- 广告位管理

### 3.2 后续迭代功能（P2）
- iOS/Android原生App开发
- 云服务器部署（从公司服务器迁移）
- 评论审核
- 举报处理
- 合同验证

## 4. 解决的核心痛点

1. **小区安全问题**：通过用户真实评价了解小区治安状况
2. **设施老化问题**：查看其他租客对小区设施的评价
3. **卫生问题**：了解小区卫生维护情况
4. **中介问题**：避雷不良中介，推荐靠谱中介
5. **房源真实性**：通过合同验证（P2）确保评论真实性
6. **押金问题**：了解房东押金退还情况

## 5. 商业模式

**当前盈利方式**：广告位变现
- 首页Banner广告
- 小区详情页广告
- 评论列表页广告
- 计费方式：CPC（0.10元/次点击） + CPM（0.05元/次展示）

## 6. 技术特点

- **多端支持**：优先开发H5和微信小程序（P0），后续支持iOS、Android（P2）
- **实时数据**：评论、评分等数据实时更新
- **高可用性**：开发阶段使用公司服务器，生产阶段支持负载均衡、异地容灾
- **数据安全**：用户隐私保护、合同信息脱敏
