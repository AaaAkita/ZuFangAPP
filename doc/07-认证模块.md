# 租房避雷App - 认证模块设计

## 1. 模块概述

**模块名称**：认证模块 (Auth Module)

**功能描述**：
- 用户注册（手机号、邮箱、微信）
- 用户登录（手机号、邮箱、微信）
- JWT Token 签发和验证
- 密码加密存储
- 登出和 Token 刷新

## 2. 认证流程

### 2.1 手机号注册流程

```
用户提交注册信息
    ↓
验证：手机号格式、密码强度
    ↓
检查手机号是否已注册
    ↓
密码加密（bcrypt）
    ↓
创建用户记录
    ↓
生成JWT Token
    ↓
返回用户信息和Token
```

### 2.2 手机号登录流程

```
用户提交登录信息
    ↓
验证：手机号格式、密码
    ↓
查询用户记录
    ↓
验证密码（bcrypt.compare）
    ↓
生成JWT Token
    ↓
返回用户信息和Token
```

### 2.3 微信小程序登录流程

```
用户点击微信登录
    ↓
前端调用 wx.login() 获取 code
    ↓
前端发送 code 到后端
    ↓
后端调用微信API：code2session
    ↓
获取 openid 和 session_key
    ↓
检查 openid 是否已注册
    ↓
    ↓ 是 → 返回已存在的用户信息和Token
    ↓    否
    ↓ 创建新用户
    ↓ 生成JWT Token
    ↓ 返回用户信息和Token
```

### 2.4 JWT 验证流程

```
客户端请求携带 Token
    ↓
JwtAuthGuard 拦截请求
    ↓
提取 Authorization Header
    ↓
验证 Token 签名和过期时间
    ↓
提取用户信息（userId）
    ↓
查询用户完整信息
    ↓
将用户信息注入请求上下文
    ↓
允许访问受保护的路由
```

## 5. 守卫设计

### 5.1 JwtAuthGuard

**功能**：验证 JWT Token，提取用户信息

**使用位置**：
- 需要登录的接口
- 管理后台接口

**实现代码**：

```typescript
@Injectable()
export class JwtAuthGuard extends AuthGuard('jwt') {
  canActivate(context: ExecutionContext): boolean | Promise<boolean> {
    const request = context.switchToHttp().getRequest();
    const token = this.extractTokenFromHeader(request);

    if (!token) {
      throw new UnauthorizedException('Token 未提供');
    }

    request.user = this.validateToken(token);
    return true;
  }

  private extractTokenFromHeader(request: any): string {
    const authHeader = request.headers.authorization;
    if (!authauthHeader) return null;
    const [type, token] = authHeader.split(' ');
    return type === 'Bearer' ? token : null;
  }

  private validateToken(token: string) {
    try {
      return this.jwtService.verify(token);
    } catch (error) {
      throw new UnauthorizedException('Token 无效或已过期');
    }
  }
}
```

### 5.2 AdminAuthGuard

**功能**：验证管理员权限

**使用位置**：
- 小区管理接口
- 广告管理接口
- 评论审核接口

**实现代码**：

```typescript
@Injectable()
export class AdminAuthGuard implements CanActivate {
  canActivate(context: ExecutionContext): boolean {
    const request = context.switchToHttp().getRequest();
    const user = request.user;

    if (!user || !user.isAdmin) {
      throw new ForbiddenException('需要管理员权限');
    }

    return true;
  }
}
```

## 6. 装饰器设计

### 6.1 @User() 装饰器

**功能**：从请求中提取当前用户信息

**使用示例**：

```typescript
@Get('profile')
async getProfile(@User() user: User) {
  return user;
}
```

**实现代码**：

```typescript
export const User = createParamDecorator(
  (data: string | undefined, ctx: ExecutionContext) => {
    const request = ctx.switchToHttp().getRequest();
    const user = request.user;

    if (!user) {
      throw new UnauthorizedException('用户未登录');
    }

    return data ? user[data] : user;
  },
);
```

## 7. 服务设计

### 7.1 AuthService

**方法列表**：

| 方法名 | 参数 | 返回值 | 说明 |
|--------|------|--------|------|
| register | RegisterDto | LoginResponseDto | 用户注册 |
| login | LoginDto | LoginResponseDto | 用户登录 |
| wechatLogin | WechatLoginDto | LoginResponseDto | 微信登录 |
| validateUser | phone, password | User | 验证用户 |
| generateToken | user | string | 生成JWT |
| refreshTokens | refreshToken | LoginResponseDto | 刷新Token |

**核心代码示例**：

```typescript
@Injectable()
export class AuthService {
  constructor(
    private prisma: PrismaService,
    private jwtService: JwtService,
  ) {}

  async login(loginDto: LoginDto): Promise<LoginResponseDto> {
    const user = await this.prisma.user.findUnique({
      where: { phone: loginDto.phone },
    });

    if (!user) {
      throw new UnauthorizedException('用户不存在');
    }

    const isPasswordValid = await bcrypt.compare(
      loginDto.password,
      user.password,
    );

    if (!isPasswordValid) {
      throw new UnauthorizedException('密码错误');
    }

    const token = this.generateToken(user);

    return {
      token,
      user: this.sanitizeUser(user),
    };
  }

  async register(registerDto: RegisterDto): Promise<LoginResponseDto> {
    const hashedPassword = await bcrypt.hash(registerDto.password, 10);

    const user = await this.prisma.user.create({
      data: {
        phone: registerDto.phone,
        password: hashedPassword,
        nickname: registerDto.nickname || `用户${registerDto.phone.slice(-4)}`,
        email: registerDto.email,
      },
    });

    const token = this.generateToken(user);

    return {
      token,
      user: this.sanitizeUser(user),
    };
  }

  private generateToken(user: User): string {
    const payload: JwtPayload = {
      sub: user.id.toString(),
      phone: user.phone,
    };

    return this.jwtService.sign(payload);
  }

  private sanitizeUser(user: User) {
    const { password, ...result } = user;
    return result;
  }
}
```

## 9. 安全设计

### 9.1 密码安全

- **加密算法**：bcrypt
- **加密强度**：10 rounds
- **密码要求**：
  - 最少 8 位
  - 最多 20 位
  - 必须包含大小写字母和数字

### 9.2 Token 安全

- **有效期**：7 天
- **签名算法**：HS256
- **密钥管理**：从环境变量读取，不提交到代码库
- **传输方式**：Header `Authorization: Bearer <token>`

### 9.3 登录限制

**待实现功能**：
- 登录失败次数限制（5次/15分钟）
- IP 限制（10次/分钟）
- 短信验证码（可选）

### 9.4 微信登录安全

- **code 有效期**：5 分钟
- **session_key 安全存储**：不暴露给前端
- **openid 绑定**：确保同一微信用户对应同一账号

| AUTH_001 | 400 | 手机号格式错误 |

| AUTH_003 | 401 | 用户不存在 |

| AUTH_005 | 401 | Token 无效或过期 |
| AUTH_006 | 403 | 需要管理员权限 |
| AUTH_007 | 409 | 手机号已注册 |
| AUTH_008 | 409 | 邮箱已注册 |
| AUTH_009 | 500 | 微信API调用失败 |
