# 租房避雷应用 - 数据库设计文档

## 文档信息
- **文档类型**：给AI Agent开发使用
- **创建日期**：2026年2月24日
- **适用范围**：数据库设计、数据库开发

---

## 1. 概述

### 1.1 数据库选型
- **数据库类型**：PostgreSQL 15
- **选择理由**：
  - 开源免费
  - 功能强大
  - 支持JSON类型
  - 支持全文搜索
  - 支持空间数据
  - 社区活跃

### 1.2 数据库架构
- **主数据库**：PostgreSQL主库（读写）
- **从数据库**：PostgreSQL从库（只读）
- **缓存数据库**：Redis（热点数据缓存）

### 1.3 数据库命名规范
- **数据库名**：小写字母，下划线分隔
- **表名**：小写字母，下划线分隔，复数形式
- **字段名**：小写字母，下划线分隔
- **索引名**：idx_表名_字段名
- **外键名**：fk_表名_字段名

### 2.1 用户表（users）

#### 2.1.1 表结构

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 用户ID |
| phone | VARCHAR(20) | UNIQUE, NOT NULL | 手机号 |

| nickname | VARCHAR(50) | | 昵称 |
| avatar | VARCHAR(255) | | 头像URL |
| gender | TINYINT | DEFAULT 0 | 性别（0未知 1男 2女） |
| birthday | DATE | | 生日 |
| province | VARCHAR(50) | | 省份 |
| city | VARCHAR(50) | | 城市 |
| district | VARCHAR(50) | | 区县 |
| role | VARCHAR(20) | DEFAULT 'user' | 角色（user/admin/moderator） |
| status | TINYINT | DEFAULT 1 | 状态（0禁用 1正常） |
| wechat_openid | VARCHAR(100) | UNIQUE | 微信OpenID |
| alipay_userid | VARCHAR(100) | UNIQUE | 支付宝UserID |
| api_key | VARCHAR(100) | UNIQUE | API密钥 |
| last_login_at | TIMESTAMP | | 最后登录时间 |
| last_login_ip | VARCHAR(50) | | 最后登录IP |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 更新时间 |
| deleted_at | TIMESTAMP | | 删除时间 |

#### 2.1.2 索引设计

```sql
CREATE UNIQUE INDEX idx_users_phone ON users(phone);

CREATE UNIQUE INDEX idx_users_alipay_userid ON users(alipay_userid);
```
