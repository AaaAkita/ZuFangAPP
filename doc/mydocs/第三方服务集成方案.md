## 2. 高德地图API集成

### 2.1 服务介绍
- **服务名称**：高德地图API
- **官网地址**：https://lbs.amap.com/
- **服务类型**：地图服务
- **用途**：地图展示、周边搜索、路径规划

### 2.2 需要申请的API

| API名称 | 用途 | 免费额度 | 超出费用 |
|---------|------|---------|---------|
| Web服务API | 地图搜索、周边查询 | 30万次/天 | 0.005元/次 |
| JavaScript API | 地图展示 | 100万次/天 | 免费 |
| Android SDK | Android地图展示 | 100万次/天 | 免费 |
| iOS SDK | iOS地图展示 | 100万次/天 | 免费 |

### 2.3 申请步骤

#### 步骤1：注册高德开放平台账号
1. 访问 https://lbs.amap.com/
2. 点击"注册"按钮
3. 填写注册信息
4. 完成邮箱验证
5. 完成实名认证

#### 步骤2：创建应用
1. 登录高德开放平台
2. 进入"应用管理" -> "我的应用"
3. 点击"创建新应用"
4. 填写应用名称：租房避雷
5. 填写应用类型：移动应用
6. 提交申请

#### 步骤3：添加Key
1. 进入应用详情
2. 点击"添加Key"
3. 选择服务平台：
   - Web端（JS API）
   - Android端
   - iOS端
4. 填写Key名称
5. 填写安全密钥（SHA1/Bundle ID）
6. 提交申请
7. 记录生成的Key和安全密钥

### 2.4 需要准备的信息

| 信息项 | 说明 | 示例 |
|--------|------|------|
| 应用名称 | 应用名称 | 租房避雷 |
| 应用类型 | 应用类型 | 移动应用 |
| 包名 | Android包名 | com.zufang.app |
| Bundle ID | iOS Bundle ID | com.zufang.app |
| SHA1 | Android签名SHA1 | XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX:XX |
| Key | API Key | xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx |
| 安全密钥 | 安全密钥 | xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx |

### 2.5 配置信息

```javascript
// 高德地图配置
const amapConfig = {
  webKey: 'YOUR_WEB_KEY',           // Web端Key
  androidKey: 'YOUR_ANDROID_KEY',   // Android端Key
  iosKey: 'YOUR_IOS_KEY',           // iOS端Key
  securityJsCode: 'YOUR_SECURITY_CODE', // 安全密钥
  apiVersion: 'v2',                 // API版本
  timeout: 5000                     // 超时时间（毫秒）
};
```

### 2.6 集成要点

#### 前端集成
- 移动端：集成高德地图SDK
- Web端：集成高德地图JavaScript API
- 地图展示：小区位置标注
- 周边搜索：两公里内设施查询

#### 后端集成
- Web服务API：周边设施搜索
