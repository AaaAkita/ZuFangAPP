# 租房避雷App - 小区模块设计

## 1. 模块概述

**模块名称**：小区模块 (Community Module)

**功能描述**：
- 小区列表查询（分页、筛选、排序）
- 小区详情查询
- 小区搜索（关键词搜索）
- 管理员添加/编辑/删除小区
- 首页推荐配置
- 高德地图位置服务集成

## 4. 服务设计

### 4.1 CommunityService

**方法列表**：

| 方法名 | 参数 | 返回值 | 说明 |
|--------|------|--------|------|
| findAll | QueryCommunityDto | PaginatedResult | 查询小区列表 |
| findOne | id | Community | 查询小区详情 |
| search | keyword, options | Community[] | 搜索小区 |
| create | CreateCommunityDto | Community | 创建小区 |
| update | id, UpdateCommunityDto | Community | 更新小区 |
| delete | id | | 删除小区 |
| updateRecommended | UpdateRecommendedDto | | 更新首页推荐 |
| getRecommended | limit | Community[] | 获取推荐小区 |
| geocodeLocation | address | {lng, lat} | 地址转经纬度 |
| reverseGeocode | lng, lat | address | 经纬度转地址 |
| getNearbyServices | lng, lat | Service[] | 获取周边服务 |

**核心代码示例**：

```typescript
@Injectable()
export class CommunityService {
  constructor(
    private prisma: PrismaService,
    private gaodeService: GaodeService,
  ) {}

  async findAll(query: QueryCommunityDto): Promise<PaginatedResult<Community>> {
    const where = this.buildWhereClause(query);
    const orderBy = this.buildOrderBy(query);

    const [items, total] = await Promise.all([
      this.prisma.community.findMany({
        where,
        orderBy,
        skip: (query.page - 1) * query.limit,
        take: query.limit,
      }),
      this.prisma.community.count({ where }),
    ]);

    return {
      items,
      total,
      page: query.page,
      limit: query.limit,
      totalPages: Math.ceil(total / query.limit),
    };
  }

  async create(dto: CreateCommunityDto): Promise<Community> {
    // 如果只有地址没有经纬度，调用高德地图获取
    if (!dto.lng || !dto.lat) {
      const location = await this.gaodeService.geocode(dto.address);
      dto.lng = location.lng;
      dto.lat = location.lat;
    }

    // 获取周边服务、地铁、商圈信息
    if (dto.lng && dto.lat) {
      const nearby = await this.gaodeService.getNearby({
        lng: dto.lng,
        lat: dto.lat,
      });
      dto.nearbyServices = nearby.services;
      dto.nearbySubway = nearby.subway;
      dto.nearbyBusiness = nearby.business;
    }

    return this.prisma.community.create({
      data: dto,
    });
  }

  async updateRecommended(dto: UpdateRecommendedDto): Promise<void> {
    // 先清空所有推荐标记
    await this.prisma.community.updateMany({
      where: { isRecommended: true },
      data: { isRecommended: false, recommendOrder: null },
    });

    // 设置新的推荐
    for (let i = 0; i < dto.communityIds.length; i++) {
      await this.prisma.community.update({
        where: { id: dto.communityIds[i] },
        data: {
          isRecommended: true,
          recommendOrder: i + 1,
        },
      });
    }
  }

  private buildWhereClause(query: QueryCommunityDto) {
    const where: any = { deletedAt: null };

    if (query.keyword) {
      where.OR = [
        { name: { contains: query.keyword } },
        { address: { contains: query.keyword } },
      ];
    }

    if (query.city) {
      where.city = query.city;
    }

    if (query.district) {
      where.district = query.district;
    }

    if (query.type) {
      where.type = query.type;
    }

    if (query.hasElevator !== undefined) {
      where.hasElevator = query.hasElevator;
    }

    if (query.roomType) {
      where.roomTypes = { has: query.roomType };
    }

    if (query.priceMin || query.priceMax) {
      where.averageRent = {};
      if (query.priceMin) {
        where.averageRent.gte = parseInt(query.priceMin);
      }
      if (query.priceMax) {
        where.averageRent.lte = parseInt(query.priceMax);
      }
    }

    return where;
  }

  private buildOrderBy(query: QueryCommunityDto) {
    const order = query.order === 'asc' ? 'asc' : 'desc';
    return { [query.sortBy]: order };
  }
}
```

## 5. 高德地图集成

### 5.1 GaodeService

**API 列表**：

| 方法名 | 参数 | 返回值 | 说明 |
|--------|------|--------|------|
| geocode | address | {lng, lat} | 地址转经纬度 |
| reverseGeocode | lng, lat | address | 经纬度转地址 |
| getNearby | lng, lat, radius | NearbyInfo | 获取周边信息 |
| searchPoi | keyword, city | Poi[] | 搜索兴趣点 |

**配置**：

```typescript
interface GaodeConfig {
  apiKey: string;           // 高德地图 API Key
  baseUrl: string;         // API 基础URL
}
```

**实现代码**：

```typescript
@Injectable()
export class GaodeService {
  private readonly baseUrl = 'https://restapi.amap.com/v3';

  constructor(private configService: ConfigService) {}

  async geocode(address: string): Promise<{lng: number, lat: number}> {
    const url = `${this.baseUrl}/geocode/geo`;
    const params = {
      key: this.configService.get('GAODE_API_KEY'),
      address,
    };

    const response = await this.httpService.get(url, { params }).toPromise();
    const { geocodes } = response.data;

    if (!geocodes || geocodes.length === 0) {
      throw new BadRequestException('地址解析失败');
    }

    return {
      lng: parseFloat(geocodes[0].location.split(',')[0]),
      lat: parseFloat(geocodes[0].location.split(',')[1]),
    };
  }

  async getNearby(lng: number, lat: number, radius = 1000): Promise<NearbyInfo> {
    // 搜索周边地铁
    const subway = await this.searchPoi('地铁站', { location: `${lng},${lat}`, radius });

    // 搜索周边商圈
    const business = await this.searchPoi('商场', { location: `${lng},${lat}`, radius });

    // 搜索周边服务
    const services = await this.searchPoi('生活服务', { location: `${lng},${lat}`, radius });

    return { subway, business, services };
  }

  private async searchPoi(keyword: string, params: any): Promise<Poi[]> {
    const url = `${this.baseUrl}/place/around`;
    const response = await this.httpService.get(url, {
      params: {
        ...params,
        key: this.configService.get('GAODE_API_KEY'),
        keywords: keyword,
      },
    }).toPromise();

    return response.data.pois || [];
  }
}
```

## 6. 控制器设计

### 6.1 CommunityController

```typescript
@Controller('communities')
export class CommunityController {
  constructor(private communityService: CommunityService) {}

  @Get()
  async findAll(@Query() query: QueryCommunityDto) {
    return this.communityService.findAll(query);
  }

  @Get('recommended')
  async getRecommended(@Query('limit') limit: number = 10) {
    return this.communityService.get.getRecommended(limit);
  }

  @Get(':id')
  async findOne(@Param('id') id: string) {
    return this.communityService.findOne(BigInt(id));
  }

  @Post('admin')
  @UseGuards(AdminAuthGuard)
  async create(@Body() dto: CreateCommunityDto) {
    return this.communityService.create(dto);
  }

  @Put('admin/:id')
  @UseGuards(AdminAuthGuard)
  async update(@Param('id') id: string, @Body() dto: UpdateCommunityDto) {
    return this.communityService.update(BigInt(id), dto);
  }

  @Delete('admin/:id')
  @UseGuards(AdminAuthGuard)
  async delete(@Param('id') id: string) {
    return this.communityService.delete(BigInt(id));
  }

  @Put('recommended')
  @UseGuards(AdminAuthGuard)
  async updateRecommended(@Body() dto: UpdateRecommendedDto) {
    return this.communityService.updateRecommended(dto);
  }
}
```

## 8. 性能优化

### 8.1 缓存策略

```typescript
// 使用 Redis 缓存热门小区
const CACHE_KEY = `communities:hot`;
const CACHE_TTL = 3600; // 1小时

async getHotCommunities(): Promise<Community[]> {
  const cached = await this.redis.get(CACHE_KEY);
  if (cached) {
    return JSON.parse(cached);
  }

  const communities = await this.prisma.community.findMany({
    where: { isRecommended: true },
    orderBy: { recommendOrder: 'asc' },
  });

  await this.redis.set(CACHE_KEY, JSON.stringify(communities), CACHE_TTL);
  return communities;
}
```

### 8.2 查询优化

- 使用索引：`(isRecommended, recommendOrder)`
- 分页查询：使用 cursor-based 分页
- 只查必要字段：避免 `SELECT *`

### 8.3 批量操作

```typescript
// 批量更新推荐状态
async batchUpdateRecommended(communityIds: number[]) {
  const updates = communityIds.map((id, index) =>
    this.prisma.community.update({
      where: { id },
      data: {
        isRecommended: true,
        recommendOrder: index + 1,
      },
    })
  );

  return Promise.all(updates);
}
```
