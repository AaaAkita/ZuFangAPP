# 租房避雷App - 搜索模块设计

## 1. 模块概述

**模块名称**：搜索模块 (Search Module)

**功能描述**：
- 全局关键词搜索（小区、评论）
- 搜索联想建议（输入时实时提示）
- 热门搜索榜单
- 用户搜索历史记录
- 热词统计（Redis）

## 2. API 设计

### 2.1 路由列表

| 路由 | 方法 | 认证 | 说明 |
|------|------|------|------|
| /search | GET | 可选 | 全局综合搜索 |
| /search/suggestions | GET | 可选 | 搜索联想建议 |
| /search/hot | GET | 可选 | 热门搜索榜单 |
| /search/history | GET | 是 | 用户搜索历史 |
| /search/history/:id | DELETE | 是 | 删除搜索历史 |
| /search/history | DELETE | 是 | 清空搜索历史 |

## 3. 数据结构

### 3.1 搜索请求 DTO

```typescript
class SearchDto {
  @IsNotEmpty()
  @IsString()
  @MinLength(1)
  @MaxLength(50)
  keyword: string;               // 搜索关键词

  @IsOptional()
  @IsIn(['北京', '天津', '河北'])
  city?: string;                 // 城市筛选

  @IsOptional()
  @IsIn(['default', 'hot', 'new'])
  sort?: string = 'default';     // 排序：默认/热度/最新

  @IsOptional()
  @IsInt()
  page?: number = 1;           // 页码

  @IsOptional()
  @IsInt()
  limit?: number = 20;          // 每页条数

  @IsOptional()
  @IsString()
  type?: string;                 // 搜索类型：community/comment/all
}
```

### 3.2 搜索结果 DTO

```typescript
class SearchResultDto {
  communities: Community[];        // 匹配的小区

  total: number;                 // 总匹配数
  page: number;                 // 当前页码
  limit: number;                 // 每页条数
}
```

### 4.1 SearchService

**方法列表**：

| 方法名 | 参数 | 返回值 | 说明 |

**核心代码示例**：

```typescript
@Injectable()
export class SearchService {
  constructor(
    private prisma: PrismaService,
    private redisService: RedisService,
  ) {}

  async search(dto: SearchDto, userId?: bigint): Promise<SearchResultDto> {
    // 记录热词统计
    await this.incrementHotKeyword(dto.keyword);

    // 记录用户搜索历史
    if (userId) {
      await this.recordHistory(userId, dto.keyword);
    }

    // 并行搜索小区和评论
    const [communities, comments] = await Promise.all([
      this.searchCommunities(dto),
      this.searchComments(dto),
    ]);

    return {
      communities,
      comments,
      total: communities.length + comments.length,
      page: dto.page,
      limit: dto.limit,
    };
  }

  private async searchCommunities(dto: SearchDto): Promise<Community[]> {
    const where: any = {
      deletedAt: null,
      OR: [
        { name: { contains: dto.keyword } },
        { address: { contains: dto.keyword } },
        { district: { contains: dto.keyword } },
      ],
    };

    if (dto.city) {
      where.city = dto.city;
    }

    let orderBy: any = {};
    if (dto.sort === 'hot') {
      orderBy = { ratingCount: 'desc' };
    } else if (dto.sort === 'new') {
      orderBy = { createdAt: 'desc' };
    } else {
      orderBy = { rating: 'desc' };
    }

    return this.prisma.community.findMany({
      where,
      orderBy,
      skip: (dto.page - 1) * dto.limit,
      take: Math.floor(dto.limit / 2),
    });
  }

  private async searchComments(dto: SearchDto): Promise<Comment[]> {
    const where: any = {
      status: 'normal',
      content: { contains: dto.keyword },
    };

    return this.prisma.comment.findMany({
      where,
      include: {
        user: {
          select: {
            id: true,
            nickname: true,
            avatar: true,
          },
        },
        community: {
          select: {
            id: true,
            name: true,
          },
        },
      },
      orderBy: { likes: 'desc' },
      skip: (dto.page - 1) * dto.limit,
      take: Math.floor(dto.limit / 2),
    });
  }

  async suggestions(keyword: string, limit: number = 10): Promise<SuggestionDto[]> {
    const communities = await this.prisma.community.findMany({
      where: {
        deletedAt: null,
        OR: [
          { name: { contains: keyword } },
          { address: { contains: keyword } },
        ],
      },
      select: {
        id: true,
        name: true,
        address: true,
        type: true,
      },
      take: limit,
    });

    // 计算匹配得分
    return communities.map(community => ({
      ...community,
      score: this.calculateScore(community, keyword),
    })).sort((a, b) => b.score - a.score);
  }

  private calculateScore(community: any, keyword: string): number {
    let score = 0;

    // 完全匹配得分最高
    if (community.name === keyword) {
      score += 100;
    }

    // 前缀匹配
    if (community.name.startsWith(keyword)) {
      score += 50;
    }

    // 包含匹配
    if (community.name.includes(keyword)) {
      score += 20;
    }

    if (community.address.includes(keyword)) {
      score += 10;
    }

    return score;
  }

  async hotSearch(limit: number = 15): Promise<HotSearchDto[]> {
    const keywords = await this.redisService.zrevrange(
      'hot_search_keywords',
      0,
      limit - 1,
      'WITHSCORES',
    );

    const result = [];
    for (let i = 0; i < keywords.length; i += 2) {
      const keyword = keywords[i];
      const score = parseInt(keywords[i + 1]);

      // 计算趋势（简单对比上一周期）
      const trend = await this.calculateTrend(keyword, score);

      result.push({
        keyword,
        searchCount: score,
        trend,
      });
    }

    return result;
  }

  async incrementHotKeyword(keyword: string): Promise<void> {
    const key = 'hot_search_keywords';
    // 使用 Redis ZINCRBY 增加分数
    await this.redisService.zincrby(key, 1, keyword);
    // 设置过期时间（30天）
    await this.redisService.expire(key, 30 * 24 * 3600);
  }

  async recordHistory(userId: bigint, keyword: string): Promise<void> {
    // 检查是否已存在
    const existing = await this.prisma.searchHistory.findFirst({
      where: {
        userId,
        keyword,
      },
    });

    if (existing) {
      // 更新最后搜索时间和次数
      await this.prisma.searchHistory.update({
        where: { id: existing.id },
        data: {
          searchCount: { increment: 1 },
          lastSearchAt: new Date(),
        },
      });
    } else {
      // 创建新记录
      await this.prisma.searchHistory.create({
        data: {
          userId,
          keyword,
          searchCount: 1,
          lastSearchAt: new Date(),
        },
      });

      // 限制历史记录数量（最多20条）
      const count = await this.prisma.searchHistory.count({
        where: { userId },
      });

      if (count > 20) {
        const toDelete = await this.prisma.searchHistory.findMany({
          where: { userId },
          orderBy: { lastSearchAt: 'asc' },
          take: count - 20,
        });

        for (const item of toDelete) {
          await this.prisma.searchHistory.delete({
            where: { id: item.id },
          });
        }
      }
    }
  }

  private async calculateTrend(keyword: string, currentScore: number): Promise<string> {
    // 对比前一天的数据
    const yesterdayKey = `hot_search_keywords:yesterday`;
    const yesterdayScore = await this.redisService.zscore(yesterdayKey, keyword);

    if (!yesterdayScore) {
      return 'stable';
    }

    const diff = currentScore - yesterdayScore;
    if (diff > 10) {
      return 'up';
    } else if (diff < -10) {
      return 'down';
    } else {
      return 'stable';
    }
  }
}
```

## 5. 控制器设计

### 5.1 SearchController

```typescript
@Controller('search')
export class SearchController {
  constructor(private searchService: SearchService) {}

  @Get()
  async search(
    @Query() dto: SearchDto,
    @User() user?: User,
  ) {
    const userId = user ? BigInt(user.id) : undefined;
    return this.searchService.search(dto, userId);
  }

  @Get('suggestions')
  async suggestions(
    @Query('keyword') keyword: string,
    @Query('limit') limit: number = 10,
  ) {
    if (!keyword) {
      return [];
    }
    return this.searchService.suggestions(keyword, limit);
  }

  @Get('hot')
  async hot(@Query('limit') limit: number = 15) {
    return this.searchService.hotSearch(limit);
  }

  @Get('history')
  @UseGuards(JwtAuthGuard)
  async getHistory(@User() user: User) {
    return this.searchService.getHistory(BigInt(user.id));
  }

  @Delete('history/:id')
  @UseGuards(JwtAuthGuard)
  async deleteHistory(
    @Param('id') id: string,
    @User() user: User,
  ) {
    return this.searchService.deleteHistory(BigInt(id), BigInt(user.id));
  }

  @Delete('history')
  @UseGuards(JwtAuthGuard)
  async clearHistory(@User() user: User) {
    return this.searchService.clearHistory(BigInt(user.id));
  }
}
```

### 6.1 热门搜索排行榜

```
Key: hot_search_keywords
Type: Sorted Set
TTL: 30天

存储格式：
member: 搜索词（如："万科"）

操作命令：
ZINCRBY hot_search_keywords 1 "万科"  // 增加搜索次数
```

## 7. 搜索算法

### 7.1 关键词匹配算法

```typescript
// 模糊匹配
function fuzzyMatch(text: string, keyword: string): number {
  // 完全匹配
  if (text === keyword) return 100;

  // 前缀匹配
  if (text.startsWith(keyword)) return 80;

  // 包含匹配
  if (text.includes(keyword)) return 60;

  // 拼音匹配（中文）
  if (pinyinMatch(text, keyword)) return 40;

  return 0;
}
```

### 7.2 搜索结果排序

```typescript
// 根据排序类型排序
function sortResults(results: any[], sortType: string): any[] {
  switch (sortType) {
    case 'hot':
      return results.sort((a, b) => b.searchCount - a.searchCount);
    case 'new':
      return results.sort((a, b) => b.createdAt - a.createdAt);
    case 'rating':
      return results.sort((a, b) => b.rating - a.rating);
    default:
      return results;
  }
}
```

### 7.3 联想建议算法

```typescript
// 基于用户历史推荐
function suggestByHistory(userId: bigint, keyword: string): string[] {
  // 获取用户搜索历史
  const history = getSearchHistory(userId);

  // 基于历史关键词推荐
  const suggestions = history
    .filter(item => item.keyword.startsWith(keyword))
    .map(item => item.keyword)
    .slice(0, 5);

  return suggestions;
}
```

## 8. 性能优化

### 8.1 缓存策略

```typescript
// 缓存热门搜索
const CACHE_KEY = `search:hot:${limit}`;
const CACHE_TTL = 1800; // 30分钟

async hotSearch(limit: number = 15): Promise<HotSearchDto[]> {
  const cached = await this.redis.get(CACHE_KEY);
  if (cached) {
    return JSON.parse(cached);
  }

  const result = await this.getHotSearchFromRedis(limit);
  await this.redis.set(CACHE_KEY, JSON.stringify(result), CACHE_TTL);
  return result;
}
```
