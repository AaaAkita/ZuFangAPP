# 租房避雷App - 评论模块设计

## 1. 模块概述

**模块名称**：评论模块 (Comment Module)

**功能描述**：
- 发布评论（避雷/推荐）
- 查询评论列表（分页、筛选、排序）
- 评论点赞和取消点赞
- 评论回复
- 评论举报
- 评论审核（管理员）
- 评论删除

## 2. API 设计

### 2.1 路由列表

| 路由 | 方法 | 认证 | 说明 |
|------|------|------|------|
| /comments | GET | 可选 | 查询评论列表 |
| /comments/:id | GET | 可选 | 查询评论详情 |
| /comments | POST | 是 | 发布评论 |

| /comments/:id/like | DELETE | 是 | 取消点赞 |
| /comments/:id/reply | POST | 是 | 回复评论 |
| /comments/:id/report | POST | 是 | 举报评论 |
| /comments/admin/:id/audit | PUT | 是 | 审核评论（管理员） |

```typescript
  @IsNotEmpty()

  @IsOptional()

  @IsIn(['避雷', '推荐'])

  @IsInt()
  rentAmount?: number;           // 租金（元/月）

  @IsOptional()
  @IsString()
  rentPeriod?: string;           // 租期

  @IsOptional()
  @IsBoolean()
```

### 4.1 CommentService

**方法列表**：

| 方法名 | 参数 | 返回值 | 说明 |

| like | commentId, userId | | 点赞评论 |
| unlike | commentId, userId | | 取消点赞 |

| report | commentId, userId, reason | Report | 举报评论 |
| audit | id, status, adminId | Comment | 审核评论 |

**核心代码示例**：

```typescript
@Injectable()
export class CommentService {
  constructor(
    private prisma: PrismaService,
    private fileService: FileService,
  ) {}

  async create(userId: bigint, dto: CreateCommentDto): Promise<Comment> {
    return this.prisma.$transaction(async (tx) => {
      // 验证文件所有权
      if (dto.images?.length > 0) {
        await this.fileService.validateFiles(dto.images, userId);
      }
      if (dto.video) {
        await this.fileService.validateFiles([dto.video], userId);
      }

      // 创建评论
      const comment = await tx.comment.create({
        data: {
          ...dto,
          userId,
          status: 'normal', // P2版本可能为'pending'
        },
      });

      // 更新小区评分和评论数
      await this.updateCommunityRating(tx, dto.communityId, dto.rating);

      return comment;
    });
  }

  async like(commentId: bigint, userId: bigint): Promise<void> {
    // 检查是否已点赞
    const existing = await this.prisma.commentLike.findUnique({
      where: {
        commentId_userId: {
          commentId,
          userId,
        },
      },
    });

    if (existing) {
      throw new BadRequestException('已点赞过该评论');
    }

    // 创建点赞记录
    await this.prisma.$transaction(async (tx) => {
      await tx.commentLike.create({
        data: { commentId, userId },
      });

      // 增加点赞数
      await tx.comment.update({
        where: { id: commentId },
        data: { likes: { increment: 1 } },
      });
    });
  }

  async report(commentId: bigint, userId: bigint, reason: string): Promise<void> {
    // 检查是否已举报
    const existing = await this.prisma.report.findFirst({
      where: {
        userId,
        targetId: commentId,
        targetType: 'comment',
      },
    });

    if (existing) {
      throw new BadRequestException('已举报过该评论');
    }

    await this.prisma.report.create({
      data: {
        userId,
        targetId: commentId,
        targetType: 'comment',
        reason,
        status: 'pending',
      },
    });
  }

  private async updateCommunityRating(
    tx: Prisma.TransactionClient,
    communityId: bigint,
    newRating: number,
  ): Promise<void> {
    const community = await tx.community.findUnique({
      where: { id: communityId },
    });

    if (!community) {
      throw new NotFoundException('小区不存在');
    }

    // 计算新的平均评分
    const newRatingAvg = (
      (community.rating * community.ratingCount + newRating) /
      (community.ratingCount + 1)
    ).toFixed(2);

    await tx.community.update({
      where: { id: communityId },
      data: {
        rating: parseFloat(newRatingAvg),
        ratingCount: { increment: 1 },
      },
    });
  }
}
```

## 5. 控制器设计

### 5.1 CommentController

```typescript
@Controller('comments')
export class CommentController {
  constructor(private commentService: CommentService) {}

  @Get()
  async findAll(@Query() query: QueryCommentDto) {
    return this.commentService.findAll(query);
  }

  @Get(':id')
  async findOne(@Param('id') id: string) {
    return this.commentService.findOne(BigInt(id));
  }

  @Post()
  @UseGuards(JwtAuthGuard)
  async create(
    @User() user: User,
    @Body() dto: CreateCommentDto,
  ) {
    return this.commentService.create(BigInt(user.id), dto);
  }

  @Delete(':id')
  @UseGuards(JwtAuthGuard)
  async delete(
    @Param('id') id: string,
    @User() user: User,
  ) {
    return this.commentService.delete(BigInt(id), BigInt(user.id));
  }

  @Post(':id/like')
  @UseGuards(JwtAuthGuard)
  async like(
    @Param('id') id: string,
    @User() user: User,
  ) {
    return this.commentService.like(BigInt(id), BigInt(user.id));
  }

  @Delete(':id/like')
  @UseGuards(JwtAuthGuard)
  async unlike(
    @Param('id') id: string,
    @User() user: User,
  ) {
    return this.commentService.unlike(BigInt(id), BigInt(user.id));
  }

  @Post(':id/report')
  @UseGuards(JwtAuthGuard)
  async report(
    @Param('id') id: string,
    @User() user: User,
    @Body() dto: { reason: string },
  ) {
    return this.commentService.report(
      BigInt(id),
      BigInt(user.id),
      dto.reason,
    );
  }

  @Put('admin/:id/audit')
  @UseGuards(AdminAuthGuard)
  async audit(
    @Param('id') id: string,
    @User() admin: AdminUser,
    @Body() dto: { status: string, remark?: string },
  ) {
    return this.commentService.audit(
      BigInt(id),
      dto.status,
      BigInt(admin.id),
      dto.remark,
    );
  }
}
```

## 6. 评论展示逻辑

### 6.1 评论展示规则

1. **排序**：默认按时间倒序，可按点赞数或评分排序
2. **分页**：每页20条，支持无限滚动
3. **筛选**：支持按类型（避雷/推荐）和标签筛选
4. **权限**：作者可以编辑和删除自己的评论
5. **状态**：只显示正常状态的评论（管理员可查看所有）

```typescript
function formatUserLocation(comment: Comment, community: Community): string {
  if (comment.isAnonymous) {
    return '匿名用户';
  }

  let location = `${community.name}`;
  if (comment.building) {
    location += `-${comment.building}号楼`;
  }
  if (comment.isVerified) {
    // 已认证用户不显示完整房间号

    location += `-${comment.unit}单元-${comment.room}`;

  return location;
}
```

```typescript
async validateFiles(fileUrls: string[], userId: bigint): Promise<void> {
  for (const url of fileUrls) {
    const file = await this.prisma.file.findFirst({
      where: {
        url,
        userId,
        type: { in: ['image', 'video'] },
      },

    if (!file) {
      throw new BadRequestException('文件不存在或无权访问');

    }
  }
```

## 8. 评论审核（P2）

### 8.1 审核状态

```typescript
enum CommentStatus {
  PENDING = 'pending',    // 待审核
  NORMAL = 'normal',      // 正常
  DELETED = 'deleted',    // 已删除
}
```

### 8.2 审核流程

```
用户发布评论
    ↓
评论状态设置为 pending
    ↓
管理员审核
    ↓
    ↓ 通过 → 状态改为 normal
    ↓    显示给用户
    ↓
    ↓ 拒绝 → 状态改为 deleted
    ↓    通知用户
```

### 8.3 审核接口

```typescript
async audit(
  id: bigint,
  status: 'normal' | 'deleted',
  adminId: bigint,
  remark?: string,
): Promise<Comment> {
  const comment = await this.prisma.comment.findUnique({
    where: { id },
  });

  if (!comment) {
    throw new NotFoundException('评论不存在');
  }

  if (comment.status !== 'pending') {
    throw new BadRequestException('评论无需审核');
  }

  const updated = await this.prisma.comment.update({
    where: { id },
    data: {
      status,
      remark,
    },
  });

  // 如果审核通过，更新小区评分
  if (status === 'normal') {
    await this.updateCommunityRating(comment.communityId, comment.rating);
  }

  return updated;
}
```

```typescript
const comments = await this.prisma.comment.findMany({
  where: { communityId },
  include: {
    user: {
      select: {
        id: true,
        nickname: true,
        avatar: true,
      },
    },
  },
  orderBy: { createdAt: 'desc' },
  skip: (page - 1) * limit,
  take: limit,
});
```

### 9.2 缓存策略

```typescript
// 缓存热门评论
const CACHE_KEY = `comments:hot:${communityId}`;
const CACHE_TTL = 1800; // 30分钟

async getHotComments(communityId: bigint): Promise<Comment[]> {
  const cached = await this.redis.get(CACHE_KEY);
  if (cached) {
    return JSON.parse(cached);
  }

  const comments = await this.prisma.comment.findMany({
    where: {
      communityId,
      status: 'normal',
      likes: { gt: 10 }, // 点赞数大于10
    },
    orderBy: { likes: 'desc' },
    take: 5,
  });

  await this.redis.set(CACHE_KEY, JSON.stringify(comments), CACHE_TTL);
  return comments;
}
```
